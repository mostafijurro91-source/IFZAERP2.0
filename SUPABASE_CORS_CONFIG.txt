============================================

Follow these steps to configure CORS in Supabase:

1. Go to: https://supabase.com
2. Login with your account
3. Select your project: "ifza-erp" (or your project name)
4. Go to: Settings → API (in left sidebar)
5. Find "CORS" section
6. Add these origins:
   - https://ifzaerp.com
   - https://www.ifzaerp.com
   - http://ifzaerp.com (development)
   - http://localhost:5173 (local development)
